## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- Python 3.11+
- Neon DB (PostgreSQL)
- Twilio account
- Groq API key
- Docker + Docker Compose

### Installation

1. **Start the full stack (recommended)**
```bash
docker-compose up --build -d
```

2. **Manual local setup (optional)**

```bash
git clone https://github.com/Gauravguddeti/relayX.git
cd relayX
```

3. **Setup Frontend**
```bash
cd frontend
npm install
npm run dev
```

4. **Setup Backend**
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

5. **Configure Environment Variables**
See `.env.example` files in both frontend and backend directories.
